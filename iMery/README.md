# iMery - 미술 작품 감상 기록 앱

iMery는 미술관에서 감상한 작품을 사진으로 촬영하고 감상평을 기록할 수 있는 웹 애플리케이션입니다.

---

## 🎨 주요 기능

### Phase 1 기능
- ✅ **헤더 네비게이션**: 로고 클릭으로 홈 이동, KO/EN 언어 토글, 알림 패널
- ✅ **저장된 작품 캐러셀**: 최대 5개 표시, 작품 클릭 시 상세 페이지, 더보기 버튼
- ✅ **태그 기반 검색**: 태그 클릭으로 검색, 여러 태그 선택 가능, 칩으로 표시
- ✅ **작품 목록 페이징**: 5개씩 로드, 더보기 버튼
- ✅ **폴더 관리**: 사용자 정의 폴더 생성, 작품 선택, 대표 이미지 지정
- ✅ **작품 상세 페이지**: 이미지, 제목, 날짜, 평점, 태그, 감상평 표시
- ✅ **최상단 스크롤 버튼**: 스크롤 시 표시, 부드러운 애니메이션

### Phase 2 기능
- ✅ **작품 수정/삭제**: 
  - 작품 페이지와 검색 결과에서 수정 버튼 제공
  - 기존 데이터 자동 입력, 원본 생성 날짜 보존
  - 삭제 확인 다이얼로그로 안전한 삭제
  
- ✅ **평점 필터 & 레이아웃 시스템**:
  - 평점 버튼 (All, 5★~1★) 공유 배경
  - 정렬 아이콘 버튼: 최신순/이름순 토글
  - 레이아웃 아이콘 버튼: 4가지 보기 모드
    - 리스트 형식 (세로 목록)
    - 큰 상자 (1열 그리드)
    - 중간 상자 (2열 그리드)
    - 작은 상자 (3열 그리드)

- ✅ **평점 필터 기능**: 메인 페이지에서 평점별 즉시 필터링

- ✅ **하단 네비게이션 개선**: 모든 페이지에서 즉시 이동, 상세 뷰 자동 초기화

- ✅ **북마크 시스템**:
  - 마우스 호버 시 북마크 아이콘 표시
  - 작품 페이지에 북마크 폴더 제공
  - localStorage로 영구 저장

## 🚀 데모 실행 방법

### 1. 사전 요구사항
- **Node.js** (v14 이상)
- **npm** 또는 **yarn**

### 2. 프로젝트 설치

```bash
# 프로젝트 디렉토리로 이동
cd /Users/apple/Desktop/React/imery

# 의존성 패키지 설치
npm install
```

### 3. 개발 서버 실행

```bash
# 개발 서버 시작
npm run dev
```

서버가 시작되면 터미널에 다음과 같은 메시지가 표시됩니다:

```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

### 4. 브라우저에서 열기

웹 브라우저를 열고 다음 주소로 이동하세요:

```
http://localhost:5173
```

### 5. 앱 사용해보기

#### 작품 등록
1. 하단의 **+** 버튼 클릭
2. 이미지 업로드 (파일 선택 또는 드래그 앤 드롭)
3. 작품 정보 입력 (제목, 카테고리, 평점, 감상평)
4. **저장** 버튼 클릭

#### 필터 및 레이아웃 변경
1. 메인 페이지에서 평점 버튼 클릭 (All, 5★~1★)
2. 정렬 아이콘 클릭 (최신순 ⇄ 이름순)
3. 레이아웃 아이콘 클릭 (리스트 → 큰 → 중간 → 작은 상자)

#### 북마크 추가
1. 작품 카드에 마우스 호버
2. 북마크 아이콘 클릭
3. **작품** 페이지의 **북마크** 폴더에서 확인

#### 작품 수정
1. 작품 카드의 **수정** 버튼 클릭
2. 정보 수정 (생성 날짜는 자동 보존)
3. **수정** 버튼 클릭

#### 작품 삭제
1. 작품 카드의 **삭제** 버튼 클릭
2. 확인 다이얼로그에서 **삭제** 버튼 클릭

#### 폴더 관리
1. **작품** 페이지로 이동
2. **새 폴더** 버튼 클릭
3. 폴더 이름 입력, 작품 선택, 대표 이미지 지정
4. **생성** 버튼 클릭

## 🛠️ 기술 스택

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v3.x
- **Icons**: Lucide React
- **Storage**: localStorage (작품, 폴더, 북마크 데이터)

## 💾 데이터 저장

앱은 localStorage를 사용하여 다음 데이터를 저장합니다:

- `imery-works`: 등록된 작품 목록
- `imery-folders`: 사용자가 생성한 폴더
- `imery-bookmarks`: 북마크한 작품 ID 목록

> **참고**: localStorage 데이터를 초기화하려면 브라우저 개발자 도구에서 Application > Local Storage를 삭제하세요.
