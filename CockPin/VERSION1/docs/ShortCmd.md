주요 단축키는 다음과 같습니다. 이 단축키 중 일부는 사용 중인 운영 체제에 따라 다를 수 있습니다.

Ctrl/⌘ + Shift + J 또는 Ctrl/⌘ + Shift + P: Cursor 전용 설정 열기
Ctrl/⌘ + Shift + I: 컨트롤 패널(Control Panel) 열기
Ctrl/⌘ + I: 컴포우저(Composer-미니 컨트롤 패널) 열기
Ctrl/⌘ + K: 코드창 인라인(Inline) 채팅
Ctrl/⌘ + L: 오른쪽 사이드바 채팅
Option/Alt + Enter: 빠른 질문 기능 (텍스트 선택 시)
Ctrl/⌘ + Enter: 채팅에서 임베딩 검색 실행

- 커서 탭을 사용하여 더 빠르게 코딩하기
- cmd+L = 채팅, cmd+i = composer, cmd+k = 편집
- 모든 프롬프트 입력 상자에서 @를 입력하여 빠르게 컨텍스트를 추가하기
- 채팅/composer에 파일을 추가하려면 + 버튼을 사용하기
- cmd+/를 사용하여 모델 간 전환
- 채팅에서 cmd+enter를 사용하여 코드베이스 전체와 채팅
- 이미지를 채팅에 드래그/드롭/복사/붙여넣기하여 사용하거나 활용하기
- composer의 경우, 코드를 테스트한 후에도 거부할 수 있도록 "모두 수락" 전에 "모두 저장"을 누르기
- cmd-shift-i를 눌러 더 큰 composer 뷰 열기
- 채팅에서 alt-enter를 눌러 현재 파일 없이 메시지를 보내기
- 채팅에서 cmd-i를 눌러 채팅 메시지를 composer로 가져옴
- @web: 채팅/composer에서 최신 정보를 얻도록 함
- @git: 변경 사항을 참조하기 위해 사용합니다. 예를 들어 커밋을 @로 지정하고 AI에게 검토를 요청
- @<link>: AI가 답변하기 전에 먼저 링크를 방문하도록 요청
- Cursor 설정 > 기능에서 문서를 추가하여 외부 문서를 @로 참조
- / 명령어를 사용하여 열린/활성 편집기를 참조
- 채팅/composer에 대한 제안 파일을 켜서 관련 파일을 더 쉽게 추가
- 모든 검색 결과를 사용하는 등 composer를 트리거할 수 있는 큐브

ursor의 Notepads 기능 사용법을 설명해드리겠습니다:
기본 조작:
새 노트패드 열기: Ctrl + N (Mac: Cmd + N)
파일 저장: Ctrl + S (Mac: Cmd + S)
모든 변경사항 저장: Ctrl + Alt + S (Mac: Cmd + Option + S)
편집 기능:
복사: Ctrl + C (Mac: Cmd + C)
붙여넣기: Ctrl + V (Mac: Cmd + V)
잘라내기: Ctrl + X (Mac: Cmd + X)
실행 취소: Ctrl + Z (Mac: Cmd + Z)
다시 실행: Ctrl + Y (Mac: Cmd + Shift + Z)
AI 지원 기능:
코드 자동 완성: 코드 작성 시 자동으로 제안됨
AI 설명 요청: 선택한 코드 우클릭 후 "Explain Code" 선택
AI 코드 수정: 선택한 코드 우클릭 후 "Fix Code" 선택
AI 채팅: 우측 패널의 채팅 기능 사용
텍스트 검색 및 이동:
찾기: Ctrl + F (Mac: Cmd + F)
바꾸기: Ctrl + H (Mac: Cmd + H)
특정 줄로 이동: Ctrl + G (Mac: Cmd + G)
다중 편집:
다중 커서: Alt + Click (Mac: Option + Click)
같은 단어 모두 선택: Ctrl + D (Mac: Cmd + D)
줄 복제: Alt + Shift + ↓ (Mac: Option + Shift + ↓)
화면 설정:
확대: Ctrl + + (Mac: Cmd + +)
축소: Ctrl + - (Mac: Cmd + -)
기본 크기로 복원: Ctrl + 0 (Mac: Cmd + 0)
특수 기능:
코드 접기/펼치기: 좌측 여백의 화살표 클릭
미니맵 표시: 우측 스크롤바 영역
들여쓰기 자동 정렬: Ctrl + Shift + I (Mac: Cmd + Shift + I)
파일 관리:
파일 열기: Ctrl + O (Mac: Cmd + O)
탭 간 이동: Ctrl + Tab (Mac: Cmd + Tab)
탭 닫기: Ctrl + W (Mac: Cmd + W)
생산성 향상 기능:
자동 저장 기능
문법 강조 표시
실시간 오류 검사
Git 통합 기능
팁:
우측 패널의 AI 채팅을 활용하여 코드 관련 질문하기
자동 완성 제안을 활용하여 빠른 코딩
다중 커서 기능으로 여러 줄 동시 편집
정기적으로 저장하는 습관 들이기
이러한 기능들을 활용하면 Cursor의 Notepads에서 효율적으로 코드를 작성하고 편집할 수 있습니다. AI 지원 기능을 적극 활용하면 생산성을 더욱 높일 수 있습니다.
